Ejercicios HackerRank

Esta tarea consistia en completar ejercicios de Java o JavaScript en HackerRank los cuales consistian en completar varías líneas de código con el objetivo de hacer que el código compilara a la perfección.

El ejercico uno titulado como "Java If Else" consistia en el uso de las declaraciones condicionales if-else para automatizar los procesos de toma de desiciones.

El ejercicio dos titulado como "Java Loops 1" consistia en hacer bucles para ayudarnos a hacer problemas matematicos simples.

El ejercicio tres titulado como "Java Loops 2" consistia en el uso de enteros para la creación de una serie especifica.

El ejercicio cuatro títulado como "Java Datatypes" consistia en el uso de los datos primitivos de Java para mantener valores enteros.

El ejercicio cinco títulado como "Java Strings Introduction" consistia en el uso y comprensión de cadenas en Java.

El ejercicio seis títulado como "Java Substring" consistia en imprimir una subcadena que consta de caracteres de un rango inclusivo desde un índice start y end.

El ejercicio siete titulado como "Java 1D Array" consistia en el uso de arrays capaces de sostener enteros.

El ejercicio ocho titulado como "Java 2D Array" consistia en imprimir la suma más grande entre todos los relojes de arena que estaban en el array.

El ejercicio nueve titulado como "Java List" consistia en el uso de listas en java en la cual se realizaban consultas y una vez completadas imprimia la lista modificada como una sola línea de enteros reparados por espacios.

El ejercicio diez titulado como "Java Sort" consistia en que nos proporcionaban una lista de información de estudiante la cual debía de reorganizarce de acuerdo un orden decreciete.

El ejercicio once titulado como "Java Comparator" consistia en crear un comparador y este se utilizaba para ordenar un array.

El ejercicio doce titulado como "Java Regex" consistia en crear una clase nueva que contiene un patrón de cadena, ya creada escribir una expresión y asignarla a un patrón de manera que pueda usarse ára validar una dirección IP.
